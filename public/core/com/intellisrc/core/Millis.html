    • com.intellisrc.core.Millis 
    • @groovy.transform.CompileStatic
class Millis
extends java.lang.Object

      Time conversion to milliseconds NOTE: MONTH is calculated as 30 days (use MONTH_31D if you want to get a 31 days month) YEAR is calculated as 365 days (use YEAR_LEAP if you want to get 366 days year)
