[Groovy] Class Version

    • com.intellisrc.core.Version 

    • @groovy.transform.CompileStatic
class Version
extends java.lang.Object

      Version object which can be used to easily update parts of it. It can import versions from String and export them to String For example: 2.0-beta1 3.11.293 4.1.1.14 5.0.1-SNAPSHOT [Static] Automatic search for version
