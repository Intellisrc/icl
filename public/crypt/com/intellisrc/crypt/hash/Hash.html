        • Field Detail

          • protected int cost

            Note: Using cost will reuse the bytes returned to rehash it again This is different than: sha(sha(x)) because sha(x) returns String HEX and not bytes (and thus restricting the characters availability). In order to imitate this method in other languages something like this must be done: sha(hex2bytes(sha(x)))
