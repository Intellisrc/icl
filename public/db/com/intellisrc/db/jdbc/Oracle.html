        • @groovy.transform.CompileStatic
class Oracle
extends JDBCServer

          Oracle Database

          Since:
          17/12/14. Additional settings: db.oracle.params = [:] NOTE: In Oracle a user is practically the same as a schema (a collection of tables, or as it is known as database in other engines). In other words, tables created by that user are allocated under its tablespace. When you drop a user you can also remove all tables associated with it.
