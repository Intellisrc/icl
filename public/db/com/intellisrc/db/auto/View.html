class View<M extends Model>
extends Relational<M>
implements com.intellisrc.etc.Instanciable<M>

        A read-only Table or View If the table or view exists (and createSQL is specified) it won't recreate the table or view unless 'recreate' is true (you can additionally specify 'DROP' statements inside the createSQL. NOTE: 'createSQL' is not required to be passed into the constructor, it can be set later, for example: MyView myView = new MyView() myView.createSQL = "CREATE VIEW my_view AS ..." boolean recreate = true if(myView.create(recreate)) { Log.i("All good!") } // Which is similar to: MyView myView = new MyView("CREATE VIEW...", true)
