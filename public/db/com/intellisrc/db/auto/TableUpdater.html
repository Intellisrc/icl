[Groovy] Class TableUpdater

    • com.intellisrc.db.auto.TableUpdater 
    • @groovy.transform.CompileStatic
class TableUpdater
extends java.lang.Object

      Updates database tables In summary, it will create a backup of a table, drop it, create it and restore the data from the backup. If the data can be inserted without having to change the data or the structure (for example, if an index was added), then you don't need to worry about passing a RecordUpdater, otherwise, you will need to check and change the data before inserting it back using such interface.
