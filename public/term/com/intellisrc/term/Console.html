Package: com.intellisrc.term

[Groovy] Class Console

    • com.intellisrc.term.Console 

    • @groovy.transform.CompileStatic
class Console
extends java.lang.Object

      A wrapper around jLine Config settings: ------------------------- console.prompt : What to place as default prompt (default: '> ' ) console.mask : What to use to cover passwords (one char, default: '*') console.timeout : If set, Trigger onTimeOut() after N seconds (default: 0) console.default : If true, will add the default commands (exit, clear, etc.) (default: true)

      18/06/27.
