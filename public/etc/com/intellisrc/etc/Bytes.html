Package: com.intellisrc.etc

[Groovy] Class Bytes

    • com.intellisrc.etc.Bytes 

    • @groovy.transform.CompileStatic
class Bytes
extends java.lang.Object

      This class contains a rewrite of some methods without using String class. The reason is that String class keep values in memory and thus poses a security risk according to some sources. While all methods in this class exists natively in Groovy/Java, those requires to change values to String, e.g: char[] password = ... bytes[] bytesPass = password.toString().getBytes() This class solves that issue, e.g: char[] password = ... bytes[] bytesPass = Bytes.fromChars(password) This class is mainly used by common.crypt package
