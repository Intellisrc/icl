        • int getContentLength()

          Parse the MJPEG frame header for the Content-Length. This value is not necessarily parsed from the MJPEG frame data because the value may not exist as a header value depending on the CameraAPI call that was used to get the stream. To retrieve the parsed value use: getProperties().getProperty(MjpegFormat.CONTENT_LENGTH)

          Returns:
          the content length of the jpeg bytes, or -1 if unknown
