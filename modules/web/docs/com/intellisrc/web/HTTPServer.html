[Groovy] Class HTTPServer

    • com.intellisrc.web.HTTPServer 

    • @groovy.transform.CompileStatic
class HTTPServer
extends java.lang.Object

      Simple HTTP server The main difference with Serviciable* classes is that this code does not depend on Spark This implementation is for basic static server and does not provide any advanced feature present in common Web Servers

      Since:
      17/07/28. based in: https://github.com/markwinter/simple-web-server/blob/master/WebServer.java
